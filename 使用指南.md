# 项目使用指南

## 项目概述

基于提示擦除策略的生成式模型投毒攻击检测研究 - 完整实现

**核心特性:**
- ✅ 3种攻击方法（BadNets、InsertSent、Syntactic）
- ✅ 4种检测策略（PromptEraser、AttentionEraser、STRIP、ONION）
- ✅ 支持多模型（LLaMA、ChatGLM、Qwen）
- ✅ 支持多数据集（SST-2、AG's News等）
- ✅ 完整的评估与可视化

## 快速开始

### 1. 环境安装

```bash
# 进入项目目录
cd project

# 安装依赖
pip install -r requirements.txt

# 安装spaCy英文模型（可选，用于句法攻击）
python -m spacy download en_core_web_sm
```

### 2. 快速测试（无需GPU）

```bash
# 运行快速测试，验证项目安装
python tests/test_quick.py
```

### 3. 运行实验

#### 方式一：使用主入口脚本（推荐）

```bash
# 攻击复现实验
python run.py --mode attack \
    --model meta-llama/Llama-2-7b-chat-hf \
    --dataset sst2 \
    --attack badnets \
    --poison-rate 0.1

# 检测对比实验
python run.py --mode detect \
    --model meta-llama/Llama-2-7b-chat-hf \
    --dataset sst2 \
    --detector prompt_eraser \
    --erase-ratio 0.3 \
    --n-iterations 10

# 参数敏感性分析
python run.py --mode sensitivity \
    --param erase_ratio

# 快速测试模式（使用小数据集）
python run.py --mode attack --quick-test --device cpu
```

#### 方式二：直接使用实验脚本


## 代码示例

### 示例1：创建攻击

```python
from src.attacks.badnets_attack import BadNetsAttack

# 创建BadNets攻击
attack = BadNetsAttack(
    trigger='cf',
    target_label=0,
    poison_rate=0.1,
    position='random'
)

# 植入触发词
text = "This movie is great"
poisoned_text = attack.inject_trigger(text)
print(poisoned_text)  # "This cf movie is great" 或类似

# 投毒整个数据集
texts = ["Text 1", "Text 2", "Text 3"]
labels = [1, 0, 1]
poisoned_texts, poisoned_labels, indices = attack.poison_dataset(texts, labels)
```

### 示例2：使用检测器

```python
from src.models.llm_wrapper import LLMWrapper
from src.detectors.prompt_eraser import PromptEraserDetector

# 加载模型
model = LLMWrapper('meta-llama/Llama-2-7b-chat-hf', load_in_8bit=True)

# 创建检测器
detector = PromptEraserDetector(
    model.model,
    model.tokenizer,
    erase_ratio=0.3,
    n_iterations=10,
    device='cuda'
)

# 检测样本
text = "This cf movie is great"
result = detector.detect(text)

print(f"是否投毒: {result['is_poisoned']}")
print(f"异常分数: {result['score']:.3f}")
print(f"置信度: {result['confidence']:.3f}")
```

### 示例3：构建ICL数据集

```python
from src.datasets.data_loader import DatasetLoader, PoisonedDataset
from src.attacks.badnets_attack import BadNetsAttack

# 加载数据
data = DatasetLoader.load('sst2', max_samples=1000)

# 创建投毒攻击
attack = BadNetsAttack(trigger='cf', target_label=0, poison_rate=0.1)

# 创建投毒数据集
poisoned_dataset = PoisonedDataset(
    data['train']['texts'],
    data['train']['labels'],
    attack=attack
)

# 构建ICL prompt
prompt, demonstrations = poisoned_dataset.create_icl_prompt(
    query_idx=0,
    n_shots=5,
    seed=42
)

print(prompt)
```

### 示例4：评估检测效果

```python
from src.evaluation.metrics import Evaluator

# 真实标签和预测标签
true_labels = [0, 0, 1, 1, 0, 1]  # 0=干净, 1=投毒

# 计算指标

print(f"FPR: {metrics['fpr']:.3f}")
print(f"FNR: {metrics['fnr']:.3f}")
```

## 参数说明

### 攻击参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| trigger | str | 'cf' | 触发词/句子 |
| target_label | int | 0 | 目标标签 |
| poison_rate | float | 0.1 | 投毒比例 |
| position | str | 'random' | 触发词位置 |

### 检测参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| erase_ratio | float | 0.3 | 擦除比例 (0-1) |
| n_iterations | int | 10 | 迭代次数 |
| aggregation | str | 'mean' | 分数聚合方法 |

### 模型参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| model | str | 'meta-llama/Llama-2-7b-chat-hf' | 模型名称 |
| device | str | 'cuda' | 计算设备 |
| load_in_8bit | bool | True | 8位量化 |

## 实验流程建议

### 第一阶段：攻击复现（3.21-4.05）

```bash
# 1. 验证攻击效果
python run.py --mode attack --dataset sst2 --attack badnets --poison-rate 0.1
python run.py --mode attack --dataset sst2 --attack insertsents --poison-rate 0.1

# 2. 不同投毒比例对比
for rate in 0.01 0.05 0.1 0.2; do
    python run.py --mode attack --poison-rate $rate
done
```

### 第二阶段：检测框架（4.06-4.15）

```bash
# 实现并测试核心检测算法
python run.py --mode detect --detector prompt_eraser --erase-ratio 0.3

# 对比基线方法
python run.py --mode detect --detector strip
python run.py --mode detect --detector onion
```

### 第三阶段：对比实验（4.16-4.25）

```bash
# 参数敏感性分析
python run.py --mode sensitivity --param erase_ratio

for model in "meta-llama/Llama-2-7b-chat-hf" "THUDM/chatglm3-6b"; do
    python run.py --mode detect --model $model
```
